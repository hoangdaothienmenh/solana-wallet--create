####  solana-wallet--create

This is project to create solana wallet. Build on nodejs and ejs template. 


### How to run:

```bash
  sh -c "$(curl -sSfL https://release.solana.com/v1.4.8/install)"
  npm i
  node server.js
  ```
  visit:  http://localhost:8080
